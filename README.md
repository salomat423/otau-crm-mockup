# OTAU CRM — кликабельный макет

Внутренняя CRM OTAU Group: ванильный фронт (HTML + CSS + ES-modules), без сборщика и бэкенда. Назначение — демонстрационная версия для презентации покупателям.

## Запуск

Из директории проекта (ES-модули требуют HTTP, не `file://`):

```bash
python3 -m http.server 8765
```

Открыть: `http://127.0.0.1:8765/`.

## Структура

- `index.html` — точка входа.
- `css/` — токены, базовые стили, layout, компоненты, ИИ-блоки.
- `js/app.js` — bootstrap.
- `js/router.js` — hash-роутинг.
- `js/shell.js` — sidebar, header, иконки навигации.
- `js/state.js` — роль, история чата, задачи.
- `js/data.js` — мок-данные: ЖК, корпуса, квартиры, менеджеры, сделки, лента deal-1, инсайты владельца, ответы ИИ-чата.
- `js/icons.js` — inline-SVG.
- `js/ui.js` — toast, modal, sparkline, escape.
- `js/screens/` — экраны: dashboard, funnel, deal, catalog, growth, aiControl, aiAdvisor, tasks.

## Демо-сценарий (5–7 минут)

1. **Открыть** `http://127.0.0.1:8765/`. Включена роль «Менеджер».
2. **Воронка** — фильтры по менеджеру/ЖК/источнику/периоду, перетаскивание карточек, ИИ-полоска снизу.
3. **Сделка** — открыть «Айгуль Нурланова»: пять блоков, лента коммуникаций с оценкой ИИ по каждой, аудио-плеер, транскрипт звонка в модалке, разбор презентации по этапам, документы и платежи.
4. **Переключить роль** в шапке на «РОП»:
   - Дашборд: сводка, рейтинг менеджеров со спарклайнами, «на грани срыва».
   - Раздел «ИИ-контроль»: команда, эталонные сценарии, спорные оценки, кнопка «Еженедельный отчёт» (модалка с PDF-превью).
5. **Переключить роль** на «Владелец»:
   - Дашборд: брифинг-баннер, KPI со спарклайнами, план/факт диаграмма, карта сделок по ЖК, эффективность каналов.
   - Раздел «ИИ-советник» (главный «вау»): лента инсайтов с приоритетами и раскрытием карточек, действия «Принять / Отложить / Не релевантно», чат с подсказками-чипами, голосовой брифинг.
6. **Каталог** → ЖК → корпус → шахматка → клик по квартире → модалка карточки → пошаговая бронь (3 шага).
7. **Задачи** — показать созданные задачи из инсайтов.

## Что внутри (соответствие ТЗ)

- CRM-веб с ролями менеджер / РОП / владелец.
- 7-этапная воронка + вкладки «Отказы / Отложенные / Постпродажа», DnD и резервное меню переноса этапа.
- Карточка сделки: профиль, параметры, лента (WhatsApp / Instagram / звонок / презентация / заметка / системные события), ИИ по каждой коммуникации, общий ИИ-блок, документы и платежи.
- Каталог: ЖК → корпус → шахматка с цветами статусов, карточка квартиры и пошаговая онлайн-бронь.
- Дашборды по ролям с реалистичными цифрами, спарклайнами и графиками.
- «Моё развитие» (менеджер), «ИИ-контроль» (РОП), «ИИ-советник» (владелец).
- Инсайты, голосовой брифинг, диалог с ИИ.

## Замечания

- Все «звонки», «аудио» и «PDF» — имитация, без реальных файлов.
- Данные в памяти, перезагрузка возвращает исходное состояние (роль сохраняется в `localStorage`).
